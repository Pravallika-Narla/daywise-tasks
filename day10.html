HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for its appearance.

p tag

The Paragraph element. The HTML element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields

pre tag

The pre tag defines preformatted text. Text in a pre tag element is displayed in a fixed-width font, and the text preserves both spaces and line breaks. The text will be displayed exactly as written in the HTML source code.

Types of tags

1.paired tags
Paired tags are used to denote content which has a clear beginning and an end.
2.void tags
There is a special group of elements that only have start tags and does not contain any content within it, these elements are called void elements. Void elements doesn't have ending tags and can only have attributes but do not contain any kind of content.

behaviour of tags

1.block level elements
A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are: p and div tags
2.inline elements
Inline elements display in a line. They do not force the text after them to a new line. An anchor (or link) is an example of an inline element. You can put several links in a row, and they will display in a line.

image tag

  • This tag plays a important role in enriching web content by embedding images into a webpage.
  • src:Specifies the path to the image.
  • alt:Provides alternate text for the image, useful for informing users about the image and displaying in case of network issues.
  • height:Specifies the height of the image.
  • width:Specifies the width of the image.

anchor tag

  • The anchor tag defines a hyperlink, which is used to link from one page to another.
  • href:This attribute, which indicates the link's destination

url structure

  • A URL (Uniform Resource Locator) is the address of a unique resource on the internet.
  • syntax:http://subdomain.domain-name.domain-extension/path-to-resource?parameters
